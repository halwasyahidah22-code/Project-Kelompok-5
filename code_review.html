BAGIAN 2: Implementasi C++ (Pemenuhan Syarat Alpro)

Aplikasi ini mengimplementasikan ke-11 elemen teknis fundamental C++ sebagai berikut:

1. Struct

Digunakan sebagai Node dan objek utama. Contoh: struct MenuItem, struct Order, struct Table.

2. References (&)

Digunakan di seluruh fungsi untuk meningkatkan efisiensi dan menghindari penyalinan memori. Contoh: const std::string& name.

3. Pointer (*)

Sangat masif digunakan untuk alokasi dinamis memori dan menghubungkan data antar Node dalam Linked List/Graph. Contoh: MenuItem* next;

4. Namespace

Logika utama dibungkus sangat rapi dalam beberapa namespace khusus: RestaurantSystem, STLUtils, Sorting, dan FileIO.

5. Callback Function

Diimplementasikan secara elegan pada fungsi applyToAllOrders(..., OrderCallback cb) untuk memproses setiap pesanan secara dinamis.

6. Default Argument / Inline Function

Fungsi-fungsi utilitas pendek dioptimalkan secara ekstensif menggunakan kata kunci inline. Contoh: inline bool menuIdExists(...).

7. Function Overloading / Template

Memanfaatkan fungsionalitas Template C++ murni untuk mencari data generik. Contoh: template<typename T> linearSearch(...).

8. Exception Handling

Program kebal dari crash berkat blok pencegahan eror yang memanfaatkan try, catch, dan pelemparan throw RestaurantException(...).

9. STL (Vector, List, Iterator, Sort, Find, Count)

Keseluruhan spektrum STL digunakan. std::vector dan std::list sebagai wadah, std::sort() dan std::find() dari <algorithm> untuk pencarian/pengurutan, serta penggunaan manual iterasi untuk logika count.

10. File Handling

Diimplementasikan di dalam namespace FileIO untuk melakukan saving dan loading data (Persistence) menggunakan pustaka bawaan ifstream dan ofstream.

11. Lambda Expression

Digunakan sebagai pembanding (comparator) anonim instan di dalam algoritma pencarian. Contoh: Lambda pada proses ekstraksi BFS std::sort(..., [](const auto& a, const auto& b){ ... }).

BAGIAN 3: Implementasi Struktur Data

Kedelapan struktur data utama sukses ditanamkan dan berjalan berdampingan (interconnected):

1. Linked List (Single/Double)

Implementasi murni Single Linked List pada MenuLinkedList (Data Master Menu) dan InventarisLinkedList.

2. Circular Linked List

Digunakan secara interaktif untuk sistem siklus. Contoh: Rotasi Meja (TableCircularList) dan Rotasi Shift Karyawan (StaffCircularList).

3. Stack (LIFO)

Implementasi tumpukan pada OrderHistoryStack untuk menghidupkan fitur Undo (membatalkan transaksi terakhir pada kasir).

4. Queue (FIFO)

Implementasi antrean antrian linear pada OrderQueue untuk mensimulasikan tumpukan pesanan yang masuk secara real-time ke dapur.

5. Binary Tree / AVL Tree

Implementasi Binary Search Tree (BST) di kelas MenuBST untuk mempercepat pencarian data menu di dalam sistem hingga O(log n), serta mencetak menu berurutan abjad via In-Order Traversal.

6. Graph dengan BFS dan DFS

Implementasi kelas bintang di aplikasi ini. Menggunakan Adjacency List untuk dua sistem besar:
- Manajemen Meja: Penelusuran rute (BFS) dan inspeksi ruangan (DFS).
- Rekomendasi Cerdas: Algoritma rekomendasi berlapis ala E-Commerce menggunakan BFS (Level 1 Neighbors) dan Pembangkit Menu Kombo via DFS.

7. Hashing dan Hash Table

Struktur data tingkat lanjut yang dieksekusi dengan std::unordered_map untuk mencari tagihan pelanggan per meja dalam waktu sangat singkat O(1), dan sebagai relasi simpul pada Graph.

8. Sorting

Algoritma pengurutan terintegrasi di banyak celah kode, seperti namespace Sorting dan std::sort(), serta sifat inheren pohon biner pada fitur kasir.
