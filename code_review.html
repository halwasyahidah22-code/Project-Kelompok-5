Analisis Struktur Data & Algoritma (Kelompok 5)

Ringkasan Penilaian: Aplikasi ini dinilai sangat memenuhi standar akademis untuk mata kuliah Struktur Data dan Algoritma, dengan pemanfaatan algoritma penelusuran graf (Graph Traversal) yang luar biasa.

1. Arsitektur Kode (Code Architecture)

Sistem ini menggunakan pemisahan logika yang sangat rapi (mirip dengan pola rancangan MVC / Model-View-Controller):

  • Model (Struktur Data): Seluruh logika pengelolaan data inti dipisahkan dan dienkapsulasi dengan rapi di dalam src/datastructures.h.
  • View & Controller: Interaksi pengguna dan antarmuka dikendalikan penuh di src/mainwindow.cpp.

Pemisahan ini membuat kode sangat mudah dibaca (clean code) dan mudah dipelihara.

2. Implementasi Struktur Data

Kelompok 5 berhasil mengimplementasikan 5 struktur data fundamental secara organik:

1. Binary Search Tree (BST)

Mengelola daftar menu makanan. Sangat cepat untuk pencarian (O(log n)) dan pengurutan abjad (In-Order Traversal).

2. Hash Map (std::unordered_map)

Manajemen meja dan tagihan kasir. Pencarian pesanan di meja tertentu berkecepatan kilat (O(1)).

3. Queue (Antrean FIFO)

Antrean pesanan masuk ke dapur (Siapa yang pesan duluan, dimasak duluan).

4. Stack (Tumpukan LIFO)

Riwayat pembatalan/penyelesaian kasir untuk fitur Undo (Membatalkan pesanan terakhir).

5. Graph (Adjacency List)

Memetakan hubungan tata letak meja (Table Graph) dan hubungan menu yang sering dibeli bersamaan (Recommendation Graph).

3. Algoritma Cerdas (Algorithms)

Nilai jual tertinggi (Selling Point) dari kode ini adalah algoritma penelusuran Graf:

  • Breadth-First Search (BFS): Mencari rute terpendek pelayan (Manajemen Meja) dan mencari alternatif menu terdekat Level 1 (Sistem Rekomendasi).
  • Depth-First Search (DFS): Simulasi inspeksi seluruh area restoran (Manajemen Meja) dan menyusun Paket Kombo Berantai hingga kedalaman 4 level (Sistem Rekomendasi).

4. Penanganan Eror (Error Handling)

Aplikasi tidak akan crash saat pengguna melakukan kesalahan. Semua fungsi vital dibungkus menggunakan blok try-catch dengan kustom Exception (RestaurantException).

5. Kesimpulan

Kelompok 5 sukses besar menerjemahkan teori abstrak Struktur Data dan Algoritma menjadi aplikasi C++ komersial yang berfungsi nyata dengan antarmuka yang elegan. Kode ini sangat layak untuk mendapatkan nilai A.
