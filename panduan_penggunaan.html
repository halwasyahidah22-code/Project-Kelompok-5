Aplikasi Restaurant System - Tugas UAS Kelompok 5

1. Dashboard Utama

Halaman pertama yang Anda lihat saat membuka aplikasi. Dashboard ini menampilkan statistik ringkas dari restoran Anda secara real-time.

  • Total Menu: Jumlah menu yang terdaftar di database.
  • Total Order: Jumlah pesanan yang sedang diproses.
  • Meja Terpakai: Jumlah meja yang sedang melayani pelanggan.

2. Manajemen Menu Binary Search Tree (BST)

Tab ini digunakan untuk mengelola daftar makanan dan minuman yang dijual.

  • Tambah Menu: Masukkan Nama Menu, Kategori, dan Harga. Tekan tombol Tambah. Sistem akan menolaknya jika nama kosong atau harga mengandung huruf.
  • Cari Menu: Ketik nama menu untuk mencarinya. Pencarian ini sangat cepat karena menggunakan algoritma struktur data BST (Binary Search Tree).
  • Tampilkan Berdasarkan Abjad: Menu otomatis diurutkan dari A-Z menggunakan metode penelusuran In-Order Traversal.

3. Penjualan & Kasir Hash Map & Queue

Tempat Anda melayani pelanggan dan mencatat pesanan.

  • Pilih Nomor Meja dari dropdown (1 sampai 8).
  • Pilih Makanan/Minuman, tentukan jumlah (Qty), lalu klik Tambah Pesanan.
  • Pesanan yang masuk akan disimpan dalam Antrean (Queue) dapur.
  • Untuk menyelesaikan tagihan meja, klik tombol Bayar & Selesaikan.

4. Riwayat Transaksi Stack (LIFO)

Semua transaksi yang sudah dibayar akan dicatat di sini.

  • Data disimpan menggunakan metode tumpukan / Stack (Last-In, First-Out).
  • Fitur ini memungkinkan Anda untuk melakukan Undo (membatalkan penyelesaian pesanan terakhir jika kasir melakukan kesalahan).

5. Manajemen Meja & Simulasi Jalur Graph BFS/DFS

Tab ini menampilkan visualisasi denah / layout meja restoran dalam bentuk Grafik (Graph).

  • Jalankan BFS: Menyimulasikan pencarian jalur terdekat pelayan dari satu meja ke meja lain secara menyebar (berlapis). Animasi akan menunjukkan warna Hijau (dilewati) dan Oranye (aktif).
  • Jalankan DFS: Menyimulasikan inspeksi kebersihan seluruh ruangan secara mendalam dari ujung ke ujung.

6. Rekomendasi Menu Cerdas Graph BFS/DFS

Sistem ini bisa merekomendasikan menu layaknya aplikasi modern (Netflix/Tokopedia) berdasarkan riwayat pembelian pelanggan sebelumnya.

  • Rekomendasi Langsung: Menampilkan menu yang paling sering dibeli secara bersamaan dengan menu pilihan Anda.
  • Rekomendasi Alternatif (BFS): Menyarankan menu berdasarkan kecenderungan (Level 1) menggunakan pencarian Breadth-First Search.
  • Paket Kombo Berantai (DFS): Membuatkan ide paket makanan lengkap berantai (maksimal 4 menu) dengan mencari jalur asosiasi terpanjang menggunakan Depth-First Search.
